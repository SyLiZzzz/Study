# Vue项目目录详解

**build** ：vue的打包环境文件

**config**：配置文件夹    **dev.env.js**开发环境的包        **prod.env.js**生存环境的包

**node_modules**：依赖包的文件夹

**src**：代码生成环境       **accets**：需要打包的静态资源   **components**：组件(虚拟页面) 

**index.html** ：唯一的页面

**App.vue**：终端入口

**router**：跳转路由的js文件夹

**static**：不需要打包的静态资源

**package.json**：类似于web项目的web.xml
